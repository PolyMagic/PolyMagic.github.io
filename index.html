My Projects

Neothesia

Neothesia is my latest Rust project Goal of this project is to bring back Legacy Synthesia that was abandoned in favour of closed source commercial project to live, and make it look and work as good (or even better) than commercial Synthesia. Neothesia is midi visualizer, which means that it takes music notes from midi file and displays them as colourful falling blocks on virtual piano.

Midi2FretBoard

Midi2FretBoard is a tool for piano players that displays notes from midi keyboard on guitar FretBoard. This allows piano players to easily learn how to play songs that they already know from piano, on a new instrument. If you are piano player and you want to quickly jump into playing guitar without searching for tabs etc, this tool is for you. I originally wrote it in GoLang but later decided to rewrite it in JS so it can be access over the web easier.

Node-MyAnimeList

Node-MyAnimeList is a promise based NodeJs package for downloading information from MyAnimeList, for simple requests like getting lists, descriptions,etc. it uses JikanAPI. I created it because in every MAL related project I always rewrote same API requests, and now they are in small convenient package. I also added some of my own scrapers into this package that enables you to remotely login into MAL, update your list and get notifications.
